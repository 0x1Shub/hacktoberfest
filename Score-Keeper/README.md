# Table Tennis Score Keeper

A simple web application for keeping track of table tennis scores. This app allows users to set a winning score, update player scores, and reset the game.

## Features

- **Set Winning Score:** Select the winning score from a dropdown (3-7).
- **Score Tracking:** Track the scores for Player One and Player Two.
- **Game Over Condition:** The game ends once a player reaches the winning score.
- **Reset Functionality:** Reset the scores to 0 and restart the game at any point.
- **Thematic Design:** Includes a table tennis image for visual appeal.
- **Responsive Layout:** The app adapts to different screen sizes, making it mobile-friendly.
